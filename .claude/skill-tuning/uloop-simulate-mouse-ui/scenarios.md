# Scenarios — uloop-simulate-mouse-ui

## A (median): Click a UI Button
> The user wants to click the `ButtonStart` element which appears at SimX=400, SimY=300 in the AnnotatedElements output. Plan the command and verification.

## B (edge): Slow drag for visual inspection
> Drag a UI slider thumb from (200, 400) to (600, 400) slowly enough that a human watching the screen could follow it. Plan the command.

## H (hold-out): Split drag — pause mid-drag for inspection
> Start a drag at (300, 300), pause to take a screenshot, move to (500, 300), pause again, then release at (700, 300). Plan all commands and the order they must run in.

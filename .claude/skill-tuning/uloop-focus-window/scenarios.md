# Scenarios — uloop-focus-window

## A (median): Focus before screenshot
> Before capturing a screenshot, bring the Unity Editor window to the foreground so the user can see what's being captured.

## B (edge): Reuse for capture-window pre-step in a workflow
> You're about to run `uloop capture-unity-window` against the Game tab. Make sure the Unity window is the frontmost window first.

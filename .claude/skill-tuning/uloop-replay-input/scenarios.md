# Scenarios — uloop-replay-input

## A (median): Replay latest recording
> The user just recorded a play session and wants to replay the latest recording exactly once. Plan the commands.

## B (edge): Loop a recording for a demo
> Set up a continuous loop replay of `scripts/intro-demo.json` so the demo cycles indefinitely. Confirm it's running.

## H (hold-out): Check progress mid-replay
> A replay is in progress. Get the current progress and report `CurrentFrame`/`TotalFrames`/`Progress` to the user.

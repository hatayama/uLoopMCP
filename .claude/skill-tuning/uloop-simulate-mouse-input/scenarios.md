# Scenarios — uloop-simulate-mouse-input

## A (median): Right-click to place a block in Minecraft-like
> The user wants to right-click at screen center (400, 300) to place a block in a game that reads `Mouse.current.rightButton`. Plan the command and follow-up.

## B (edge): FPS camera look — pan smoothly right
> Pan the FPS camera 300 pixels to the right over 0.5 seconds (smooth, human-like). Plan the command.

## H (hold-out): Project uses old Input Manager, not New Input System
> The user wants to click at (400, 300) but their project uses the legacy Input Manager, not the New Input System. Decide what to do and report.

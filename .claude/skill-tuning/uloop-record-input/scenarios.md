# Scenarios — uloop-record-input

## A (median): Capture WASD play
> The user wants to record a 5-second WASD movement run in PlayMode for later replay. Plan the commands and what you'll report back.

## B (edge): Save to a specific path
> Record the user's input but save it to `scripts/regression-bug-42.json` so it can be checked into the repo for a bug-reproduction case.

## H (hold-out): Recording started but Unity left PlayMode mid-recording
> A recording is in progress and the user accidentally stopped PlayMode. They ask you to stop the recording and explain what happened.

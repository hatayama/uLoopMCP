# uloop get-version

Get uLoopMCP version information.

## Usage

```bash
uloop get-version
```

## Parameters

None.

## Examples

```bash
# Get version info
uloop get-version
```

## Output

Returns JSON with version information:
- uLoopMCP package version
- CLI version

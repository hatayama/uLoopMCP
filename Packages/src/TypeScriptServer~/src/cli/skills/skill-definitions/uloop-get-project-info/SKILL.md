# uloop get-project-info

Get detailed Unity project information.

## Usage

```bash
uloop get-project-info
```

## Parameters

None.

## Examples

```bash
# Get project info
uloop get-project-info
```

## Output

Returns JSON with project information:
- Unity version
- Project name
- Platform settings
- Build target
- Other project metadata
